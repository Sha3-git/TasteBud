## Goals & Objectives

### Project Goals
The project will aim to create a allergy and food tracker that will simplify the process for figuring out what a persons allergies and food intolerances are.

### Project Objectives
The objectives that are needed to achieve this goal include:
* Being able to track both food and symptoms throughout the day.
* Foods and symptoms being associated with one another.
* Pattern detection to be able to track the associations.
* Cross reactivity detection.
* Being able to show the data to the user and/or doctors in a way that makes sense and is accurate.

## Overall Project Risks
The risks that are currently known with the project are:
* Third party data.
* Incorrect user data leading to false positives/incorrect symptom assumptions.
