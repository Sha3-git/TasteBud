TasteBud

A clean, modern toolkit for managing food allergies — designed for clarity and confidence.

Introduction

TasteBud helps you identify and manage allergens and food intolerances in your daily food routine so you can eat freely and safely.

API Reference

The REST API provides endpoints for food data, allergens, and user preferences. Designed to be fast, predictable, and secure.

GET /api/allergens
POST /api/users
GET /api/foods?safe=true

Usage

  1. Create your TasteBud account.
  2. Add allergens and dietary restrictions.
  3. Scan food barcodes or upload ingredients.
  4. Receive instant alerts for unsafe foods.
